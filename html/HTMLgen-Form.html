Class Form - Define a user filled form. Uses POST method.

SYNOPSIS

import HTMLgen
class Form
  def __init__(self, cgi = None, **kw)
  string __module__ = 'HTMLgen'
  def __str__(self)
  def append(self, *items)

DESCRIPTION

cgi is the URL to the CGI processing program. Input objects (any others as well) are appended to this container widget.

def append(self, *items)

Append any number of items to the form container.

Keywords

name
name of the form
submit
The Input object to be used as the submit button. If None (default) a Submit button will automatically be appended to the form. Do not manually append your submit button. HTMLgen will append it for you.
reset
flag indicating if a Reset button should automatically be appended to the form.
target
set a TARGET attribute
enctype
specify an Encoding type.
onSubmit
script, which is executed, when the form is submitted
