Function fixsize

SYNOPSIS

import imgfix
def fixsize(imgmatch)

DESCRIPTION

fixsize(imgmatch) imgmatch is a match object containing an IMG tag. Will return a new IMG tag containing verified height and width properties. All other properties are left alone.
