Function find_docstring

SYNOPSIS

import HTMLutil
def find_docstring( s, begin=0)

DESCRIPTION

return (b,e) s.t. s[b:e] is the leftmost triple-quoted string in s (including the quotes), or (None, None) if s doesn't contain a triple-quoted string
